## 설명

한글 TTS를 만들기 위해 약간의 작업을 해두었습니다.

데이터셋만 만들면 나머지는 버튼만 누르면 될 정도로 쉽게 커스텀 해두었습니다.

데이터셋 예제는 다음과 같습니다.

싱글 speaker 기준
```
dataset/130061007_kr.wav|안녕히 가세요
```

원신 데이터 셋은 첨부해둔 파일의 크롤링으로 얻을 수 있습니다.

학습은 구글 Colab에서 하는 것을 추천합니다.
